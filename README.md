# Broomhandle Game!

a browser game by Victor Lawrence, made for UGA Hacks 11.

Fly around on a broomstick as Byte, the magical mascot dog!

Play: [Broomhandle.tech](https://broomhandle.tech)

This is a SvelteKit Project, therefore it requires Node.js in order to build and run. 

## Music

The background music in this game is Sketchbook 2025-11-12.ogg by Abstraction
